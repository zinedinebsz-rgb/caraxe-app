## 3. Optimisation UX "Conversion First"

### Contraste et Lisibilité Mobile

**Problèmes identifiés** :
- `--text-dim: oklch(62% 0.012 50)` sur `--bg: oklch(14% 0.008 50)` → ratio ~5.5:1 ✓ OK pour corps
- `--text-mid: oklch(76% 0.012 55)` → ratio ~8:1 ✓ Bon
- **Points faibles** : Les tags de section (petite taille + couleur dim) sont difficiles à lire sur mobile
- **Gold dim** `oklch(60% 0.09 85)` sur fond sombre → ratio ~4.8:1 → limite WCAG AA pour texte courant

**Recommandations** :
1. Remonter `--text-dim` à `oklch(66% 0.012 50)` pour gagner 0.5 point de ratio
2. Remonter `--gold-dim` à `oklch(65% 0.09 85)` pour les prix et labels dorés
3. Sur mobile, augmenter le `font-size` des section-tags de `--text-xs` à `--text-sm`
4. Ajouter un `text-shadow: 0 1px 3px oklch(0% 0 0 / 0.5)` sur le hero pour lisibilité sur fond complexe

## 4. Audit Technique de Contenu

### Trust Section — Haut de Page

**Paragraphe de confiance à insérer sous le hero** :

> CARAXES est un agent de sourcing basé physiquement en Chine — à Yiwu et Guangzhou, au cœur des zones industrielles. Nos tarifs sont affichés sur ce site : commission de 3 à 8% selon le volume, aucun frais caché, premier devis toujours gratuit. Chaque lot est inspecté en vidéo avant expédition. Plus de 150 commandes livrées depuis 2024, zéro litige non résolu.

**Pourquoi ça fonctionne** : La cible (commerçants français, souvent d'origine maghrébine ou subsaharienne) a été échaudée par les "agents" Alibaba qui disparaissent après paiement. Ce paragraphe répond immédiatement aux 3 objections : « Vous êtes qui ? », « Combien ça coûte vraiment ? », « Comment je sais que c'est fiable ? ».


### Stratégie d'Harmonisation des Prix

**Problème** : Le site affiche des prix en section Tarifs, mais les formulaires de devis ne reprennent pas ces prix. Un prospect qui lit "3% à 8%" puis reçoit un devis avec une structure différente perd confiance.

**Solution en 4 points** :

1. **Uniformiser la grille** : Sur le site ET dans les devis envoyés par email/WhatsApp, utiliser exactement la même structure :
   - Formule Création : X% (commission sur la valeur FOB)
   - Formule Gestion : X% + frais QC
   - Formule Formation : forfait fixe

2. **Ajouter un disclaimer clair** sous chaque prix affiché :
   > « Tarif indicatif. Le devis final dépend du volume, de la complexité du sourcing et de la destination. »

3. **Simulateur = même logique** : Le calculateur d'économies doit utiliser la même base de calcul que les vrais devis

4. **Devis PDF** : Générer automatiquement des devis PDF (via le dashboard admin) qui reprennent le même design que le site — même police, mêmes couleurs, même structure tarifaire
